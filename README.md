## 🔭 Projects In Depth

### 🖥️ ObscuraOS
A proprietary operating system built independently from the ground up, focused on privacy, encryption, and secure computing. ObscuraOS does not rely on any existing OS base and is designed to:
- Prioritize strong, end-to-end encryption
- Provide a secure, self-contained local computing environment
- Support development and programming tools natively

### 🔒 ZeroVault
An encrypted local notes application built for users who take their data privacy seriously. All notes are stored locally with strong encryption — no cloud, no tracking, no compromise.

### 🧠 Obscura Engine
An AI-powered research tool that searches and aggregates information from multiple online sources, including developer platforms, forums, and open-source knowledge bases. Designed for deep technical research aggregation with AI search capabilities.

### 🏠 Home Automation & Security System
A full-featured smart home system built around the **ESP32** microcontroller, capable of controlling and monitoring household infrastructure including lighting, security, and environmental systems.

### 🤖 Robotic Arm
A robotic arm prototype built using **Arduino** and **3D printing**, featuring:
- A custom controller built in **Unreal Engine**
- **Computer vision**-based hand gesture control via OpenCV
